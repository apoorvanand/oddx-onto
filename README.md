oddx-ontology
=============


Generate spec with:

      python2 specgen6.py --indir=.../oddx-onto 
        --ns=http://purl.org/openddx/oddx.rdf# --prefix=oddx 
        --ontofile=oddx.rdf --outfile=oddx.html 
        --templatedir=.../oddx-onto/style

Note: both `rdfs:label` and `rdfs:comment` must be set.
